## 1XX: informacionais
- muito pouco usados  
## 2xx: sucesso
- 200 Ok
- 201 Created
- 202 Accepted
- 204 No Content
## 3xx: redirecionamento
- pouco usado em REST
- 301 Redirected: comum em websites
## 4xx: erro de client
- 400 Bad Request
- 401 Unauthorized
- 403 Forbidden
- 404 Not Found
- 405 Method Not Allowed
## 5xx: erro de server
- 500 Internal Server Error
- 501 Not Implemented
- 502 Bad Gateway
- 504 Gateway Timeout
